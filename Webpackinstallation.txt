to make changes to this program navigate to the project directory inside a terminal and install webpack with the following command. 
(Node has to be installed)

	> npm i -D webpack webpack-cli

Once installation is finished run the command 
	> NPM Run Build
This will build th e project into the bundle.js and can show changes you made in main.js
